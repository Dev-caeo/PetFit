# 🐾 PetFit - Controle de Alimentação Pet

Aplicação web desenvolvida com foco em estudo de **UX/UI** e prototipagem, utilizando ferramentas de IA do Google AI Lab.

## 📌 Sobre o projeto

O **PetFit** é um sistema para controle alimentar de pets, permitindo acompanhar dados importantes de forma visual e intuitiva.

A proposta é ajudar donos de pets a manterem a saúde dos seus animais através de um acompanhamento simples e acessível.

## 🚀 Funcionalidades

* Controle de calorias consumidas
* Monitoramento de calorias gastas
* Indicador visual de alimentação (semicírculo de consumo)
* Cálculo de índice corporal do pet (IMC)
* Interface focada em usabilidade

## 🎯 Objetivo

Desenvolver uma interface moderna e funcional aplicando conceitos de UX/UI, com foco em:

* Simplicidade
* Clareza visual
* Feedback ao usuário
* Facilidade de uso

## 🧠 Tecnologias e Ferramentas

* TypeScript
* Figma (prototipação)
* Google AI Lab
* Material Design 3
* Conceitos de Design Thinking

## 🎨 Design e UX

O projeto foi desenvolvido com base em boas práticas de usabilidade, incluindo:

* Heurísticas de Nielsen
* Feedback visual por cores (verde → vermelho)
* Layout limpo e minimalista
* Componentização baseada em Material Design

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
